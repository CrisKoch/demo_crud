Quem somos
Projeto Sistema Web MVC e SQL

Filas estão presentes em nossas rotinas, seja para pagar uma conta ou receber um dinheiro. É fato que ninguém enfrenta uma fila porque quer e espera que chegue sua vez o mais rápido possível. Isso fez com que nos dedicássemos a minimizar os problemas e efeitos da espera e irritação dos clientes. Para tanto, desenvolvemos um sistema web monolítico utilizando template engine MVC e acesso a dados baseado em comandos SQL. A concepção deste projeto foi para que o empresário tenha facilidade para organização, utilizando uma tecnologia de template engine, e o cliente tenha a comodidade de aguardar como preferir, sendo avisado pelo colaborador momentos antes de seu atendimento.

Tecnologias utilizadas neste sistema:

  • GitHub
  • Maven
  • Spring boot
  • MySQL
  • Javascript
  • JQuery
  • MVC
  • Bootstrap 4
